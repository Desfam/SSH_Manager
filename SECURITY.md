# Security Policy

If you discover a security vulnerability, please contact the repository owner privately via GitHub Security or email: desfam@users.noreply.github.com (or open a GitHub Security Advisory).

Do not include secrets, passwords, or private keys in issues or PRs.

Reporter guidance:
- Provide a short description and reproduction steps.
- Indicate affected versions and relevant logs (avoid sending secrets).
- If needed, request a private communication channel.
